# MiMo英语课堂 - 安装部署说明

## 文件清单
```
mimo-pwa/
├── index.html      ← 主程序文件
├── manifest.json   ← App配置（名字、图标等）
├── sw.js           ← 离线缓存（让App无网也能用）
├── icon-192.png    ← App图标（你需要自己准备，见下方说明）
├── icon-512.png    ← App图标大图（你需要自己准备）
└── 安装说明.md     ← 本文件
```

---

## 第一步：准备App图标

图标需要两个尺寸的PNG图片：
- `icon-192.png`（192×192像素）
- `icon-512.png`（512×512像素）

**免费制作方法：**
1. 去网站：https://www.canva.com（免费）
2. 新建192×192的设计
3. 写上"MiMo"或者放一个📚图
4. 导出为PNG
5. 再做一个512×512的
6. 放到这个文件夹里

---

## 第二步：上传到GitHub Pages（免费）

### 注册GitHub账号
1. 打开 https://github.com
2. 点右上角 Sign up
3. 填邮箱、密码、用户名

### 创建仓库
1. 登录后点右上角 **+** → **New repository**
2. Repository name 填：`mimo-english`
3. 选 **Public**（必须选Public才能免费用Pages）
4. 点 **Create repository**

### 上传文件
1. 在新建的仓库页面点 **uploading an existing file**
2. 把文件夹里所有文件拖进去（index.html、manifest.json、sw.js、两个图标）
3. 拉到下面点 **Commit changes**

### 开启Pages
1. 仓库页面点上方 **Settings**
2. 左边菜单找 **Pages**
3. Source 选 **Deploy from a branch**
4. Branch 选 **main**，点 **Save**
5. 等1-2分钟，刷新页面就能看到网址

**你的网址会是：**
```
https://你的用户名.github.io/mimo-english
```

---

## 第三步：苹果手机安装方法

把网址发给苹果用户，告诉他们：

1. 用 **Safari浏览器** 打开网址（必须是Safari，其他浏览器不行）
2. 点底部的 **分享按钮**（正方形加箭头的图标）
3. 向下滚动找到 **"添加到主屏幕"**
4. 点右上角 **"添加"**
5. 完成！桌面出现MiMo英语图标

## 常见问题

**Q：更新了内容怎么让用户也更新？**
A：把 sw.js 第一行的 `mimo-english-v1` 改成 `mimo-english-v2`，重新上传，用户下次打开会自动更新

**Q：能改名字吗？**
A：在 manifest.json 里改 name 和 short_name 就行

**Q：苹果用户能收到通知吗？**
A：iOS 16.4以后支持，需要额外配置，之后可以加
